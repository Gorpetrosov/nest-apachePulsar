
## Installation
for more information visit` [Apache Pulsar web site](https://pulsar.apache.org/docs/3.2.x/getting-started-docker-compose/)

```bash
$ pnpm install
```
```bash
$ docker-compose up -d
```
## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
